Statcast Visualization

MLB Statcast Project

This is a project revolving around the gathering, transforming, and presenting of statistics using MLB's Statcast technology, from the 2015 season onward. My overarching goal is to find trends within this data as to how the game is evolving, find underrated players, and uncover patterns and interesting information about the game of baseball.

This project makes use of Python, SQL, PowerBI, Pandas, and Bigquery.
