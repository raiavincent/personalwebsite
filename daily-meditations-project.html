Daily Meditations Project

Project created to dissseminate a passage from Marcus Aurelius's The Meditations on a daily basis using Python. The main module that is used to run the program is EZGmail, developed by Al Sweigart, to send the passage from a gmail account set up for this specific purpose. Next and also important is the schedule module, which easily allows the code to run on set days and time, so the passage is there in the morning. Run on a Raspberry Pi 4.
